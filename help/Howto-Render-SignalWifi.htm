This is the Signal/Wifi Renderer

Her you can render Signal or Wifi icons. There are a few styles to choose from (dropdownbox above).
So far it only supports 5 Levels from 0 to 4. So if your ROM has more levels. (Some Samsung Roms have 6 levels) don't use this!

These renderes borrow a few rom-settings that the battery icons use. Like battery icon height! This makes sure these icons match the battery icons in height!
In the secont RomSettings Tab you find naming patterns for the icons.

Learn how to find the right Rom Settings

Overview

The overview shows all rendered icons in a grid. This overview is stored as a png to disc every time you press the play-button!
This overvies and the battery png's are stored in a subfolder of the Rom Fumbler:

  • ./pngs/renderer/wifi/<style-name>/...
  • ./pngs/renderer/signal/<style-name>/...

So if you want the png's only and no flashable zip...then look in that subfolder(s)!
